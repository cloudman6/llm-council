# Multi-LLM Collaborative Debate System

多大模型协同讨论与共识生成系统


## 项目目标

本项目旨在构建一个工具，使多个大型语言模型（LLM）能够围绕同一问题进行：
- 互相启发
- 批评讨论
- 信息整合
- 共识形成

最终由系统自动输出**高质量、稳定、结构化的“综合结论”**。

该系统适用于：
- 高难度推理
- 多角度决策
- 策略规划
- 产品研发讨论
- 复杂问题分析

## 系统结构设计

本系统总共有 四个 LLM 角色：

角色            职责
llm             参与讨论，从自己的角度提出观点
chairman        主持人，总结、评估、判断是否收敛


## 核心思想：发散 → 收敛

系统采用两阶段设计：

### 第一轮：发散讨论（Divergent Phase）
- LLM 按顺序发言，可以看到前面的其他LLM的观点
- 目的是最大化观点差异、产生启发
- 讨论流程：
llm1 → llm2（看见 llm1） → llm3（看见 llm1 & llm2）→ ...

Chairman 在第一轮结束时：
- 总结观点：共同点和冲突点
- 判断是否达成共识
- 如未收敛，生成下一轮问题
- 收敛则输出最终答案

### 第二轮及以后：收敛（Convergent Phase）

- 每个 LLM 不再看到其他 LLM 的回答
- 只基于 chairman 提供的总结与问题进行回应
- 目的是逐步减少差异、对齐观点、形成稳定结论

Chairman 在每轮结束：
- 总结观点：共同点和冲突点
- 判断是否达成共识
- 如未收敛，生成下一轮问题
- 收敛则输出最终答案
